# gamefic

Base inicial clonada do projeto Dermative para o novo site da Gamefic.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

O projeto esta configurado para publicar no GitHub Pages usando a base `/`.
Para usar um subdominio customizado como `gamefic.isabellavieirasouto.com`, configure esse dominio em `Settings > Pages` no GitHub e aponte um `CNAME` no Cloudflare para `isabellavsa94-cmd.github.io`.
